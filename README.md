# Protein-Molecular-Weight-Calculator
A bioinformatics tool developed in Python to calculate the molecular weight of protein sequences based on standard amino acid residue masses.
# Protein Molecular Weight Calculator

## Overview
This project calculates the molecular weight of a protein sequence using Python.

## Features
- Accepts protein sequences
- Calculates molecular weight
- Uses standard amino acid masses

## Technologies Used
- Python
